.NET Fast Admin (NFAdmin)

## 介绍
> NFAdmin 是一个开源项目，使用Bootstrap和AdminLTE为用户界面，使用ASP.NET MVC 构建的快速管理系统开发。

+ 设计数据库表结构
+ 一键生成后台管理框架和各表的CRUD界面和代码

## UI
- AdminLTE

## 服务端开发
- ASP.NET MVC
- ASP.NET Core

## 支持的数据库
- SqlServer
- MySql

## 默认功能
- 用户管理
- 角色管理
- 权限管理
- 功能管理
- 站内消息
- 站内聊天
- 邮件模板管理

## 特性
- 国际化（多语言）
- UI换肤
- 一键生成CRUD，支持Excel导出，导入
